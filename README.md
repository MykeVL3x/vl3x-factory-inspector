**Inspect VL3X factory presets differently!**

An interactive browser-based tool for exploring all 258 factory presets and their 422 parameters on the TC-Helicon VoiceLive 3 Extreme.

## Features

- **259 Factory Presets** - All presets (1-254) plus Global presets (490-494)
- **422 Parameters Per Preset** - Complete parameter data for each preset
- **Setup Configuration** - Global settings (Input, Output, Guitar, MIDI, Tone, Mix)
- **Category Filtering** - Vocal, Guitar, Mixer, HIT Control, Global, Looper
- **Search** - Filter presets by name or number
- **Zero Dependencies** - Works offline in any modern browser

## Parameter Categories

### Preset Parameters (per preset)

| Category | Parameters | Description |
|----------|------------|-------------|
| Vocal | 183 | Harmony, Delay, Reverb, HardTune, Transducer, Voice 1-8 |
| Guitar | 116 | Amp Style, EQ, MicroMod, Delay, Reverb, Effects |
| Mixer | 40 | Level and Width for all effect blocks |
| Globals | 25 | Effect on/off switches |
| HIT Control | 23 | HIT button parameter mappings |
| Looper | 17 | Loop track info and trigger slots |

### Setup Configuration (global)

| Category | Parameters | Description |
|----------|------------|-------------|
| Input | 11 | Input type, Mic gain, Lead delay, Pitch correct |
| Output | 6 | XLR level/mode, USB trim, Guitar jacks mode |
| Guitar | 5 | Phase, Gate threshold, Speaker sim, Input gain |
| MIDI | 9 | Channels, Filters, Transpose, Pitchbend, Tempo |
| Tone | 21 | Tone EQ, Adaptive mode |
| Mix | 11 | Main levels, Headphone mix, Effects mix |

## Usage

Open `index.html` in any modern browser. No server required.

1. **Browse** - Scroll through presets in the left panel
2. **Search** - Type to filter by preset name or number
3. **Select** - Click a preset to view its parameters
4. **Explore** - Use category tabs to navigate parameter groups
5. **Setup** - Click the Setup button to view global configuration
