Building VL3X Control using MIDI & SysEx

Control your VL3X from onSong, an MC-8, or any MIDI controller. Note: You can't load or modify a preset that hasn't been initialized.

A. Switch to a Preset

Load a preset using standard MIDI messages. No SysEx required.

1

For Presets 1-128

Send Program Change only. Value = preset number.

Example for Preset 50: PC=50

2

For Presets 129+

Send Bank Select first, then Program Change.

  • CC 0: Value = 0
  • CC 32: Value = preset รท 128 (rounded down)
  • PC: Value = preset % 128

Example for Preset 300: CC0=0, CC32=2, PC=44

B. Change Parameters Live (Temporary)

Modify any parameter in real-time. Changes are temporary and lost when you switch presets.

1

Load Your Preset

Use the Program Change method from Section A.

2

Send Editor Enable

F0 00 01 38 00 6D 15 00 F7
3

Delay/Wait 200ms

4

Send Parameter Change

Use the SysEx Generator to build the command. Repeat for multiple params (50ms between).

C. Store Changes Permanently

Use With Caution

Stored changes overwrite the preset. This is rarely needed for live performance.

1-4

Follow Section B Steps

5

Send Store Command

F0 00 01 38 00 6D 24 00 F7

Timing Summary

  • After preset switch (PC)200ms
  • After Editor Enable200ms
  • Between parameter changes50ms
  • After Preset Store500ms
