How to Use

  1. Enter a preset number (optional)
    • Leave blank to modify the current live preset
    • Enter 1-500 to load a preset
  2. Add parameters to your sequence
    • Select Category → Subcategory → Parameter
    • Set the value (use Option dropdown for named values)
    • Click + Add to Sequence
    • Repeat for each parameter you want to change
  3. Generate and copy
    • Click Generate Sequence
    • Click Copy to Clipboard
    • Paste into onSong, MC-8, or your MIDI controller
Saving to preset is likely not something you want to do during a live set. Consider the save carefully if you do use it.
